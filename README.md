# 🔌 ESP32 Microcontroller – Bluetooth Communication (Embedded C)

This project demonstrates **Bluetooth communication using the ESP32 microcontroller**, programmed in **Embedded C**.  
It covers initialization, pairing, data transmission, and reception between ESP32 and external devices (smartphones, PCs, or other microcontrollers).

---

## ⚙️ Overview

The ESP32 is a powerful SoC featuring:
- Dual‑core Xtensa LX6 processor
- Integrated Wi‑Fi and Bluetooth (Classic + BLE)
- GPIO, UART, SPI, I²C, PWM, ADC/DAC peripherals

This project uses **Embedded C** to configure and control Bluetooth functionality for data exchange.
